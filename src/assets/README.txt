# Flutter Lesson — Remotion Project
# FlutterByMousa · Instagram Reels Pipeline

Place your voiceover MP3 file here and name it:

  voiceover.mp3

Remotion's staticFile() helper resolves paths relative to this
assets/ folder (configured as publicDir in remotion.config.ts).

For each lesson in the n8n pipeline, swap this file out before rendering.

## Naming convention (suggested)
  lesson_01_voiceover.mp3  →  rename to voiceover.mp3 before render
  lesson_02_voiceover.mp3  →  rename to voiceover.mp3 before render
  …
